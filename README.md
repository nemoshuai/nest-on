
## Description

nestJS学习demo

## 项目搭建
1.安装全局依赖
```bash
npm -g @nestjs/cli
```

2.在目录下新建项目

```bash
nest new [project-name]
cd [project-name]
npm run start
或
npm run start:dev （watch模式，热更新）
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
